Mantuko by chmaruni

a new programming-language (sort of)

Mantuko is a combination of a programming paradigm (Faces), a concurrent programming technique (explicit scheduling), and maybe other stuff I find useful (to be done).

Is Mantuko a Programming Language?

Mantuko was supposed to become a new programming language. However, not surprisingly it turned out that it makes so much more sense to adapt an existing language instead of designing a new one from scratch. And since Smalltalk is so amazingly flexible, it can be adapted without changing the language or the VM really. Therefore, Mantuko is not a real language but rather a Smalltalk library.

Dependencies

Mantuko is implemented in GNU Smalltalk. (I use version 3.2)

Install

Having GNU Smalltalk installed, you can just type rake install. This will create a GNU Smalltalk 'star' file and place it in the ~/.st folder. From then on you can use PackageLoader fileInPackage:'Mantuko' in your Smalltalk programs.

To run the tests (without installing the package), type rake test
