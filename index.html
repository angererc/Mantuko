Mantuko by Christoph Angerer

a framework for analyzing and–hopefully some time–compiling parallel programs

Mantuko is a project that aims at analyzing fine-grained parallel programs. Until further notice, this project is rather incomplete, though; so do not expect to compile the sources and get anything useful out of it yet.

Dependencies

Mantuko is written in Erlang R13B04 and uses make. There could be a dependency to the Lathos log server, but you have to check that on your own.

Install

Having Erlang installed, download or check out the source and type one of "make", "make doc", "make test".
